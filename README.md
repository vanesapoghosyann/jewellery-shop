# Jewellery Shop

A multi-page e-commerce website built with HTML, CSS, and JavaScript based on a UI/UX design created in Figma.

This project was developed as part of my front-end development learning journey and involved transforming a complete design into a fully functional website. During the development process, I collaborated with a UI/UX designer, implementing the provided layouts and user flows while focusing on responsiveness, usability, and attention to detail.

The website simulates a real online jewellery store and includes product browsing, individual product pages, shopping cart functionality, search features, user authentication interfaces, feedback forms, and other interactive elements. The project allowed me to strengthen my skills in responsive web development, DOM manipulation, JavaScript functionality, and translating design concepts into working user interfaces.
